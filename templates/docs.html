Начало

В блоке HEADER каждого запроса обязательно должны быть отправлены следующие HTTP заголовки:

Название Тип Описание Пример
X-API-KEY String Ключ клиента documentation

Методы

Создать

Метод для создания нового объявления

Параметры запроса:

Название Тип Описание Обязательно
userId Integer ID пользователя подавшего объявление Да

Параметры ответа:

Название Тип Описание Nullable
id String Уникальный идентификатор Нет

Пример запроса:

{
    "userId": 1
}

Пример ответа:

{
    "id": "3564b135da536107"
}

Отправить запрос:

{ "userId": 1 }
